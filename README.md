# node-spider
获取 [my.ishadowx.net](https://my.ishadowx.net/) 上的ssr二维码并下载到本地，仅供练习使用node.js写爬虫。

## Techs

1. https模块
2. cheerio模块
3. fs模块

## How to use

1. yarn install
2. yarn start
3. get images from img folder
4. yarn run clean (empty the img folder)

## Notes

由于网络原因，不是每次都能下载成功的。
